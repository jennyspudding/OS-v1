## PWA Features

This application is now a fully functional Progressive Web App (PWA) with the following features:

### âœ… Installable
- Users can install the app on their devices (mobile, tablet, desktop)
- Install prompt appears automatically when PWA criteria are met
- Custom install prompt component for better user experience

### âœ… Offline Support
- Service worker caches static assets for offline functionality
- Intelligent caching strategies for different resource types:
  - **CacheFirst**: Fonts, audio, video, static JS assets
  - **StaleWhileRevalidate**: Images, CSS, regular JS files
  - **NetworkFirst**: API calls and dynamic content

### âœ… App-like Experience
- Standalone display mode (no browser UI when installed)
- Custom splash screens for iOS devices
- Theme color integration
- App shortcuts for quick access to menu and cart

### âœ… Cross-Platform Icons
- Multiple icon sizes (192x192, 512x512)
- Maskable icons for better Android integration
- Apple touch icons for iOS
- Windows tile configuration

### âœ… Apple iOS Support
- Comprehensive splash screen support for all iOS devices
- Apple-specific meta tags for web app behavior
- Touch icon optimization

## Technical Implementation

### Dependencies
- `next-pwa`: PWA plugin for Next.js
- `pwa-asset-generator`: Automated icon and splash screen generation

### Key Files
- `next.config.ts`: PWA configuration with next-pwa
- `public/manifest.json`: Web app manifest
- `public/browserconfig.xml`: Windows tile configuration
- `public/icons/`: Generated app icons and splash screens
- `components/PWAInstallPrompt.tsx`: Custom install prompt
- `app/layout.tsx`: PWA meta tags and configuration

### Service Worker
- Automatically generated by next-pwa
- Precaches static assets
- Runtime caching for dynamic content
- Workbox-powered for reliability
